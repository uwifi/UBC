# UBC 
Ubiquitous Business Chain   
普惠商业价值链，简称：商链   

# 使命 
致力于构建更加友善和强大的商业应用区块链基础架构。  
 
#技术路线   
基于Node.js, docker，共识：POW，深度学习运算工作量证明机制（我们仍然在考虑PoS是否可行，事实上的选择应该是以应用场景为主要考虑）。 
实际上区块链并不仅仅是一种技术，其中包含的哲学思想，是UBC设计的更重要的依据。
